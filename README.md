## Quick Start

Preview Link : https://neog-noter.netlify.app

## Run Locally

Clone the project

```bash
  git clone https://github.com/nerdyspook/noter.git
```

Go to the project directory

```bash
  cd noter
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```

## Features Included

-   Authentication:
    -   User Signup
    -   User Login
-   Notes Page
    -   Add a Note
    -   Update a Note
    -   Delete Note
-   Filter by Label Page
-   Search Note by Title
-   Archives
    -   Add note to Archive
    -   Restore note from Archive
    -   Delete note from Archive
-   Pin / Unpin a note
-   Trash Page
    -   Restore from Trash
    -   Delete from Trash Permanently
